# Design : UI/UX cosmétique + i18n (EN/FR/ES)

Date : 2026-05-26  
Plugin : timetracker (GLPI 11)

---

## Contexte

Le plugin affiche des données de suivi temps dans trois endroits : l'onglet "Time budget" sur un contrat, l'onglet "Contract time" sur un ticket, et le dashboard Outils > Contract time. L'UI actuelle utilise `tab_cadre_fixe` avec des éléments trop serrés et sans indicateurs visuels de statut. Aucune traduction n'existe en dehors du fallback anglais.

## Périmètre

- **UI** : améliorations cosmétiques uniquement (Option A) via classes Bootstrap 5 embarquées par GLPI 11 — aucun fichier CSS custom, aucun JS ajouté
- **i18n** : EN (fallback natif), FR (`fr_FR`), ES (`es_ES`) via fichiers gettext `.po`/`.mo`

---

## Section 1 : Onglet "Time budget" (contrat)

### Bloc de synthèse (lecture seule)

- Trois métriques (Initial / Consumed / Remaining) dans une `<div class="row p-3 mb-3">`
- Chaque métrique dans `<div class="col-md-4 text-center">`
- Progress bar Bootstrap :
  - Conteneur : `<div class="progress mb-2" style="height:20px">`
  - Barre : `<div class="progress-bar bg-{success|warning|danger}">`
  - Pourcentage calculé : `min(100, round(spent / initial * 100))`
  - Couleur : `bg-success` si OK, `bg-warning` si threshold atteint, `bg-danger` si dépassement
- Badge statut : `<span class="badge bg-{success|warning|danger}">` avec label traduit

### Formulaire (écriture)

- Conteneur : `<div class="p-3">`
- Layout 2 colonnes : `<div class="row g-3">`
- Chaque champ dans `<div class="col-md-6">`
- Labels avec `<label class="form-label fw-semibold">`
- Bouton Save : `btn btn-primary`

---

## Section 2 : Dashboard (Outils > Contract time)

### Tableau principal

- Tableau `tab_cadre_fixe` conservé
- Cellules : `class="p-2"` ajouté sur chaque `<td>`
- Après chaque ligne contrat : ligne supplémentaire avec progress bar thin (`height:8px`) sur toute la largeur
- Badges statut : `badge bg-success/warning/danger`
- Ligne Total : `class="table-secondary fw-bold"`

### Bloc dernières saisies

- Séparé par `<div class="mb-4">`
- Cellules avec `class="p-2"`

---

## Section 3 : Onglet "Contract time" (ticket)

### Structure

Trois blocs distincts séparés par `<div class="mb-4">` :

1. **Bloc total** : `<div class="p-3 mb-4">` avec total en `fw-bold`
2. **Formulaire ajout** : `<div class="p-3 mb-4">`, champs en `row g-3 col-md-6`, bouton `btn btn-primary` avec icône `ti ti-plus`
3. **Historique** : tableau avec `p-2` sur les cellules, bouton delete `btn btn-sm btn-outline-danger` avec icône `ti ti-trash`
